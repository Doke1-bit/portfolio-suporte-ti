## Sobre este portfólio

Este repositório reúne documentação técnica produzida com base em experiência prática de administração de infraestrutura em nuvem. Os documentos seguem formatos reais utilizados em ambientes corporativos de TI: estudos de caso, tutoriais, FAQs e POPs (Procedimentos Operacionais Padrão).

Todo o conteúdo é baseado em projetos reais executados de forma independente, com foco em AWS EC2, Windows Server e suporte técnico.

---

## 📁 Documentos

### 🔍 Estudo de Caso — Servidor Minecraft na AWS
`estudo-de-caso-servidor-minecraft-aws.md`

Administração de instância EC2 com Windows Server para hospedagem de servidor de jogos com até 30 usuários. Inclui diagnóstico de incidente real de desempenho e conectividade, com identificação de causa raiz e solução implementada.

**Competências demonstradas:** troubleshooting, análise de logs, otimização de recursos, AWS EC2, Windows Server.

---

### 📖 Tutorial — Criação de Instância EC2 com Windows Server
`doc1-tutorial-ec2-windows.md`

Guia passo a passo para provisionar e acessar uma instância EC2 Windows na AWS do zero. Inclui configuração de Security Groups, recuperação de senha e checklist de verificação final.

**Competências demonstradas:** documentação técnica, AWS EC2, RDP, Security Groups, boas práticas de segurança.

---

### ❓ FAQ — Suporte Técnico AWS EC2 e Windows Server
`doc2-faq-suporte-aws.md`

Base de conhecimento com 10 perguntas e respostas organizadas em 4 categorias: conectividade, desempenho, segurança e custos. Formato pronto para uso em ferramentas como Zendesk, Freshdesk ou Confluence.

**Competências demonstradas:** base de conhecimento, suporte N1/N2, diagnóstico de problemas, AWS, Windows Server.

---

### 📋 POP-001 — Reinicialização e Verificação de Instância EC2
`doc3-pop-reinicializacao-ec2.md`

Procedimento Operacional Padrão completo com 3 fases, checklist de pré-requisitos, tabela de registro de execução e tratamento de exceções. Formato compatível com ambientes que seguem ITIL.

**Competências demonstradas:** elaboração de POPs, ITIL, administração de infraestrutura, documentação operacional.
